**Project Requirements**
**Use proper Architectural Components or Android Jetpack Components. For
example, View Model, Live Data, Data Binding etc. Use MVVM.
Job Specification Information:**
1. Create a new Android Studio project and name it UserProfileRegistration
2. Add necessary dependencies-
3. Add necessary permissions for use local storage in Manifest file
4. Create an WelcomeActivity, this will be your launcher Activity.
5. Create ProfileListActivity. Later, you have to show all profile list.
6. Add a edit button for each profile.
7. Add a button for delete profile.
8. Show the total number profile in profile list activity
9. Create a AddProfileActivity. Here user will input his all information
10. Create a singleProfileActivity. Here you will show single user profile.
